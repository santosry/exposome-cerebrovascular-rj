# Data Directory

This repository is intentionally lightweight.

- `raw/`: raw DATASUS, INMET and SIDRA/IBGE downloads are regenerated by scripts and ignored by Git.
- `interim/`: intermediate files are regenerated by scripts and ignored by Git.
- `processed/`: processed analytical datasets are regenerated by scripts and ignored by Git.
- `external/`: optional external resources are ignored by Git.

Only `.gitkeep` placeholders are retained here so the public repository preserves its expected structure.
