# Methodological Overview

This repository contains the epidemiological DLNM axis of the project. The analysis estimates nonlinear and delayed associations between macroregional climate exposure and cerebrovascular outcomes in Rio de Janeiro using daily time series from 2010 to 2025.

The current public version uses only:

- DATASUS SIH-RD/SIM-DO outcome data;
- INMET temperature and relative humidity data;
- SIDRA/IBGE population denominators for the offset.

Only the epidemiological-climate workflow is included in this release.

The study is ecological and does not claim individual-level causal inference.
