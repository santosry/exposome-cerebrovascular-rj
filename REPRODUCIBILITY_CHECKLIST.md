## Data Processing

- [x] Cerebrovascular outcomes restricted to ICD-10 I60-I69
- [x] I60-I64 retained as sensitivity definition
- [x] CID subtype stratification: I60-I62 (hemorrhagic), I63 (ischemic)
- [x] Environmental exposure assigned by municipality of residence → macroregion
- [x] INMET stations mapped by municipality name + centroid fallback
- [x] Missing climate values filled by nearest macroregion
- [x] Population offset from SIDRA/IBGE, aggregated to macroregion-year
- [x] PM2.5 coverage audited: 92/92 municipalities, 74 directly monitored, 18 assigned by nearest monitored neighbor
- [x] PM2.5 temporal granularity audited: annual, optional linear covariate only

## Model Parameters

- [x] DLNM lag horizons: 7, 14, and 21 days
- [x] Main lag horizon: 14 days
- [x] Exposure spline df: 4 (sensitivity grid: 3, 4, 5)
- [x] Lag spline df: 3 with log-knots (sensitivity grid: 3, 4, 5)
- [x] Time trend: natural spline with 7 df/year
- [x] Day-of-week, holidays, and pandemic period included
- [x] Complementary exposure adjusted (ns, df=3)
- [x] Influenza control: ns(influenza_lag7, df=2) when available
- [x] PM2.5: disabled by default; optional annual linear sensitivity term via `DLNM_ENABLE_AIR_QUALITY=true`
- [x] MMT-based centering for cumulative RR

## Bayesian Stage

- [x] Input: cumulative log(RR) estimates and standard errors from DLNM
- [x] Model: normal-normal hierarchical (empirical Bayes, grid-search)
- [x] Output: posterior mean, posterior SD, 95% credible interval, Pr(RR > 1.10)
- [x] Prior sensitivity: Sceptical, Optimistic, Flat priors
- [x] Purpose: stabilization of regional estimates (shrinkage)

## Sensitivity Analyses

- [x] Lag maximum sensitivity: 7, 14, 21 days
- [x] Spline df sensitivity: 3×3 grid (df_exp × df_lag)
- [x] Pandemic exclusion sensitivity (2020-03 to 2022-12)
- [x] Prior sensitivity (Bayesian stage)
